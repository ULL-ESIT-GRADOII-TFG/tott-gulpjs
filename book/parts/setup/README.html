Setup

The first step is to get the boilerplate for this tutorial from github. Make sure you have git installed.

The boilerplate repository for this tutorial is located at: https://github.com/Coalman/tott-gulpjs. You can either clone the repository or fork it. Run the following command in a shell to clone the repo:

$ git clone https://github.com/Coalman/tott-gulpjs.git

Once it's finished, you should have a directory named tott-gulpjs in your current directory.
